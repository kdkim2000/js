# PRD — ko.javascript.info 크롤러 & 학습 사이트

## 1. 배경 및 목적

[ko.javascript.info](https://ko.javascript.info)는 모던 자바스크립트 튜토리얼의 한국어 번역 사이트다.
공식 사이트는 광고와 외부 스크립트가 포함되어 있어 집중적인 학습 환경을 제공하지 않는다.
이 프로젝트는 사이트 전체를 크롤링해 로컬에서 빠르고 쾌적하게 읽을 수 있는 자체 학습 사이트를 구축하는 것이 목적이다.

## 2. 사용자 및 사용 시나리오

- **주 사용자**: JavaScript를 학습 중인 개발자 (단일 사용자 로컬 환경 가정)
- **시나리오 1**: 오프라인 또는 빠른 읽기 — 네트워크 없이 로컬에서 아티클을 읽는다.
- **시나리오 2**: 검색 — 특정 개념(예: "클로저", "프로미스")을 전문 검색으로 즉시 찾는다.
- **시나리오 3**: 순서 학습 — 파트→챕터→아티클 순서대로 이전/다음 탐색으로 진도를 관리한다.
- **시나리오 4**: 코드 실행 — 아티클 내 예제 코드를 브라우저에서 바로 실행해 확인한다.

## 3. 기능 요구사항

### 3.1 크롤러 (Crawler)

| ID | 요구사항 | 우선순위 |
|----|---------|---------|
| C-01 | 사이트 전체 목차(TOC)를 파싱해 JSON으로 저장한다 | P0 |
| C-02 | 각 아티클 본문을 Markdown으로 변환해 파일로 저장한다 | P0 |
| C-03 | 아티클에 포함된 이미지를 로컬에 다운로드한다 | P0 |
| C-04 | SQLite + FTS5로 전문 검색 인덱스를 생성한다 | P0 |
| C-05 | 이미 크롤링된 아티클은 스킵한다 (증분 크롤링) | P1 |
| C-06 | 요청 간 딜레이 및 지수 백오프 재시도를 적용한다 | P1 |
| C-07 | 배치 분할로 병렬 크롤링을 지원한다 (`--batch=N`) | P1 |
| C-08 | 실패한 아티클 목록을 `errors.json`으로 기록한다 | P2 |

### 3.2 사이트 (Site)

| ID | 요구사항 | 우선순위 |
|----|---------|---------|
| S-01 | 홈 페이지에서 파트/챕터/아티클 전체 목차를 표시한다 | P0 |
| S-02 | `/[slug]` 경로로 개별 아티클을 렌더링한다 | P0 |
| S-03 | 헤더 검색바에서 FTS5 전문 검색을 제공한다 | P0 |
| S-04 | 아티클 내 코드 블록에 Shiki 문법 강조를 적용한다 | P0 |
| S-05 | 이전/다음 아티클 내비게이션을 제공한다 | P1 |
| S-06 | 좌측 사이드바로 전체 목차 탐색을 제공한다 | P1 |
| S-07 | localStorage로 읽음/안읽음 진도를 추적한다 | P1 |
| S-08 | 브라우저 내에서 JavaScript 코드를 직접 실행할 수 있다 | P2 |
| S-09 | 다크모드를 지원한다 | P2 |

## 4. 비기능 요구사항

- **성능**: 아티클 페이지 로드 < 200ms (정적 빌드 기준)
- **크롤링 예의**: 요청 간 최소 1.2초 딜레이, User-Agent 명시
- **오프라인**: 빌드 이후 외부 네트워크 없이 동작
- **확장성**: 단일 사용자 로컬 환경, 서버 배포 불필요

## 5. 범위 외 (Out of Scope)

- 사용자 계정/인증
- 멀티 사용자 / 서버 배포
- 원본 사이트 콘텐츠 수정
- 자동 재크롤링 스케줄러 (현재 수동 실행)
